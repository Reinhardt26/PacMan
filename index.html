ABOUT

Welcome to PACMAN with a South African twist, this is a PACMAN game I coded using Javascript

What is the South African twist you may ask, well in this PACMAN game the objective is to collect electricity bolts, since PACMAN is currently experiencing the dreaded LOADSHEDDING.

The objective is to avoid the ghosts because when it is dark the ghosts can kill you.
However, if you can reach the solar panel power up you will have a few seconds of power,
once the power is on, the ghosts become scared and PACMAN will be able to catch them, be careful though because as soon as the lights go off the ghosts will not be scared anymore.

Collect all the electricity bolts to win the game.
Have fun playing

-Reinhardt Burger

Controls

Movement

- Use W key to move up
- Use A key to move left
- Use S key to move down
- Use D key to move right

Buttons

The start button begins a new game
The pause button pauses the game
The resume button resumes the game
